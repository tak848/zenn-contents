# プロダクト概要

このリポジトリは、Zenn に投稿する記事・本を `zenn-cli` で管理するためのコンテンツ管理リポジトリです。主に「執筆 → プレビュー → 推敲 → 公開」を、再現可能な開発フローとして回すことを目的にしています。

## コア機能（パターン）

- `zenn-cli` による記事/本の作成とローカルプレビュー
- Markdown + YAML フロントマターを前提としたコンテンツ管理
- `textlint` による日本語文章の品質チェック（表記ゆれ・文体混在・文長など）
- `images/` 配下に画像アセットを整理し、記事から参照する運用
- `pnpm`（必要に応じて `aqua`）でツールチェーンを固定し、環境差を減らす

## 想定ユースケース

- 新規記事の下書き作成、公開前の調整（`published: false` で管理）
- 画像つき記事の執筆（スクリーンショット等を `images/` で管理）
- 公開前の文章校正（`textlint` 実行をゲートにする）
- Publication への投稿を含む運用（必要に応じてフロントマターで指定）

## 価値

- コンテンツを「コードのように」差分管理し、執筆フローを手元で再現できる
- 文章品質チェックを自動化し、公開直前の手戻りを減らす
- 画像・メタデータを含めて一箇所で管理でき、記事の保守性が上がる
